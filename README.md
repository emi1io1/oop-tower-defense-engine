# OOP Tower Defense Engine

A Python tower defense game simulation where ants defend tunnels from invading bees.

## Features
- Full object-oriented game architecture
- Multiple unit types with unique behaviors
- Turn-based simulation engine
- Status effects and interactions

## Concepts Demonstrated
- Inheritance and polymorphism
- State management
- Simulation design
- Large multi-class architecture

## Run
python3 ants.py
